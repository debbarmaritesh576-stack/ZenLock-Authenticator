
# ZenLock

A modern and secure Android Authenticator app built with Kotlin and Jetpack Compose.

## Features
- TOTP 2FA Codes
- QR Code Scanner
- Biometric Lock
- Offline Secure Storage
- Material 3 UI

## Tech Stack
- Kotlin
- Jetpack Compose
- Hilt
- Room Database
- CameraX
